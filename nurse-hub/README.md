# Nurse Hub
    
    Nurse Hub is a professional platform designed for healthcare education and certification preparation.
    
    ### 🛠 Tech Stack
    * **Frontend:** React, TypeScript, Tailwind CSS
    * **Backend:** Node.js, Express
    * **Database:** PostgreSQL with Drizzle ORM
    * **DevOps:** Docker, Nginx
    
    ### 🚀 Key Features
    * **Feature 1:** Comprehensive study materials and practice tests.
    * **Feature 2:** Performance tracking and analytics.
    
    ### 📁 Project Structure
    * `/client`: Frontend source code.
    * `/server`: Backend API logic.
    * `/shared`: Shared types and logic.
